# Wind Energy Prediction Using Artificial Neural Networks (ANN)

## Overview

This project predicts wind energy output using an Artificial Neural Network (ANN). The model learns the relationship between weather-related features and wind energy generation to make accurate predictions.

## Objectives

* Predict wind energy output using machine learning.
* Apply Artificial Neural Networks to a regression problem.
* Evaluate model performance using RMSE and R² Score.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* TensorFlow / Keras

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. ANN Model Development
5. Model Training
6. Model Evaluation
7. Result Visualization

## Model Performance

| Metric         | Value  |
| -------------- | ------ |
| Train RMSE     | 5.10   |
| Test RMSE      | 5.34   |
| Train R² Score | 0.9104 |
| Test R² Score  | 0.9037 |

## Results

The ANN model achieved over 90% R² on the test dataset, demonstrating strong predictive performance and good generalization. The Actual vs Predicted analysis showed that the model successfully captured the relationship between weather conditions and wind energy output.
